# ぐらぐらアニマルタワー

ゲームジャム提出用のローカル交代制ミニゲームです。1台のPCで最大4人が順番に、手番開始時に決まった動物を積みます。

## 起動

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
npm run preview
```

## ゲーム概要

- 1台4人までの交代制ローカル対戦。
- 手番開始時に次の動物が固定されます。
- マウス / ←→ で位置を調整します。
- ホイール / Q / E で角度を調整します。
- Q / E は長押しで連続回転します。
- クリック / SPACE で落とします。
- 落ちたらHP -1。HPが0になると脱落します。
- 一巡ごとに追加どうぶつが降り、盤面が荒れます。
- RESULTでは順位、スコア、勝因、内訳を表示します。

## 今回のゲームジャム向け調整

- Steamストア用素材ではなく、会場で伝わる見た目と操作説明に寄せました。
- TITLEをゲームジャム展示向けの大きなポスター画面へ変更しました。
- HOW TOを4枚に整理しました。
- GAME HUDに現在スコア、固定動物、RISK、COMBO、残り時間を表示しました。
- 開始時にREADY演出を追加しました。
- マウス移動で動物が変わるバグを修正済みです。
